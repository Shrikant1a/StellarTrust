# Contributing to Trustlance

Thank you for your interest in contributing to Trustlance! We're building the future of decentralized escrow on Stellar, and we'd love to have your help.

## 🚀 How to Contribute

There are many ways to contribute, including:

- **Reporting Bugs**: Open an issue describing the bug and how to reproduce it.
- **Suggesting Features**: Have an idea? Share it in an issue!
- **Improving Documentation**: Fix typos, clarify explanations, or add new guides.
- **Code Contributions**: Fix bugs or implement new features.

## 🛠️ Getting Started

1. **Fork the repo** and create your branch from `main`.
2. **Install dependencies**: `npm install`.
3. **Run locally**: `npm run dev`.
4. **Make your changes** and ensure everything is working.
5. **Write tests** (if applicable).
6. **Submit a Pull Request** with a clear description of your changes.

## 📜 Development Guidelines

- Follow the existing code style.
- Keep components focused and reusable.
- Use semantic HTML and descriptive variable names.
- Ensure your changes are accessible.
